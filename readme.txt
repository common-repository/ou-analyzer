== Description ==

OU ANALYZER is the plugin which displays number of users, posts, pages and comments. Also, the plugin can display information in the last 24 hours. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPres.
1. Go to Settings --> OU ANALYZER in WordPress admin area.
